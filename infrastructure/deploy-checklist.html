RankerToolAI — Deploy Checklist

Hướng dẫn từng bước để website live và SEO đúng chuẩn. Làm theo thứ tự.

PHASE 1 — Server & Domain (Ngày 1)

1.1 Mua Domain & Hosting bắt buộc

1
Mua domain rankertoolai.com

Dùng Cloudflare Registrar (rẻ nhất, không markup) hoặc Namecheap. Bật Privacy Protection.

2
Thuê VPS hoặc hosting

Khuyến nghị: Hetzner Cloud CX21 (~€5/tháng, EU datacenter) hoặc DigitalOcean Droplet 1GB (~$6/tháng). Cài Ubuntu 22.04.

3
Upload website lên server

Copy toàn bộ thư mục html/ lên /var/www/rankertoolai/html/

rsync -avz ./html/ user@your-server:/var/www/rankertoolai/html/

4
Cài nginx + cấu hình

Copy infrastructure/nginx-site.conf lên server:

sudo cp nginx-site.conf /etc/nginx/sites-available/rankertoolai.com

sudo ln -s /etc/nginx/sites-available/rankertoolai.com /etc/nginx/sites-enabled/

sudo nginx -t && sudo systemctl reload nginx

5
Cài SSL certificate (HTTPS)

sudo apt install certbot python3-certbot-nginx

sudo certbot --nginx -d rankertoolai.com -d www.rankertoolai.com

Auto-renew đã được cài tự động bởi Certbot.

6
Trỏ DNS về server

Trong Cloudflare DNS, thêm A record:

TypeNameContentProxy
A@IP_SERVER_CUA_BANProxied (cam)
AwwwIP_SERVER_CUA_BANProxied (cam)

Bật Cloudflare proxy (cam) để có CDN miễn phí + DDoS protection.

PHASE 2 — Google Analytics 4 (Ngày 1)

2.1 Tạo GA4 Property bắt buộc

1
Vào analytics.google.com

→ Admin → Create Property → chọn "Web" → nhập rankertoolai.com

2
Lấy Measurement ID

Admin → Data Streams → chọn stream → copy ID dạng G-XXXXXXXXXX

3
Thay vào main.js

Mở html/assets/js/main.js, tìm dòng:

var GA_ID = 'G-XXXXXXXXXX';

Thay G-XXXXXXXXXX bằng ID thật của bạn.

4
Xóa GA snippet cũ trong index.html (nếu có)

Mở html/index.html, xóa đoạn <script> GA cũ trong <head> vì main.js đã handle rồi.

5
Test bằng GA4 DebugView

Cài Chrome extension "Google Analytics Debugger" → mở website → kiểm tra DebugView trong GA4.

PHASE 3 — Google Search Console (Ngày 1-2)

3.1 Verify domain & Submit Sitemap bắt buộc

1
Vào search.google.com/search-console

→ Add Property → chọn "Domain" → nhập rankertoolai.com

2
Verify qua DNS TXT record

Copy TXT record Google cung cấp → thêm vào Cloudflare DNS → Done.

TypeNameContent
TXT@google-site-verification=XXXXXX
3
Submit Sitemap

Search Console → Sitemaps → nhập https://rankertoolai.com/sitemap.xml → Submit

4
Request Indexing cho trang homepage

URL Inspection → nhập https://rankertoolai.com/ → "Request Indexing"

Làm tương tự cho 5 review pages quan trọng nhất.

PHASE 4 — Bing Webmaster Tools (Ngày 2)

4.1 Setup Bing nên làm

1
Vào bing.com/webmasters

→ Add a Site → nhập https://rankertoolai.com

2
Import từ Google Search Console

Bing cho phép import sitemap và settings trực tiếp từ GSC — tiết kiệm thời gian.

3
Submit Sitemap

Sitemaps → nhập https://rankertoolai.com/sitemap.xml

PHASE 5 — Cloudflare Settings (Ngày 1-2)

5.1 Performance & Security nên làm

1
Bật Automatic HTTPS Rewrites

SSL/TLS → Edge Certificates → bật "Always Use HTTPS" + "Automatic HTTPS Rewrites"

2
Bật Speed optimizations

Speed → Optimization → bật Auto Minify (JS, CSS, HTML) + Brotli compression

3
Bật Rocket Loader (cẩn thận)

Speed → Optimization → Rocket Loader: TEST trước — đôi khi ảnh hưởng JS.

4
Cài Page Rules cho redirect www

Rules → Page Rules → www.rankertoolai.com/* → "Forwarding URL 301" → https://rankertoolai.com/$1

PHASE 6 — PageSpeed Audit (Ngày 2-3)

6.1 Kiểm tra Core Web Vitals bắt buộc

1
Chạy PageSpeed Insights

pagespeed.web.dev — test cả Mobile và Desktop. Target: Mobile ≥ 80, Desktop ≥ 95.

2
Kiểm tra LCP (Largest Contentful Paint)

Target: < 2.5s. Nếu chậm → thêm fetchpriority="high" vào logo image trong header.

<img src="/assets/images/logo-dark.png" fetchpriority="high" width="38" height="38">

3
Kiểm tra CLS (Cumulative Layout Shift)

Target: < 0.1. Thêm width và height attribute vào tất cả <img> tags để tránh layout shift.

4
Test trên WebPageTest.org

webpagetest.org → chọn location Singapore (gần VN nhất) → test 3 lần → xem waterfall.

PHASE 7 — Formspree (Contact & Newsletter)

7.1 Kết nối Contact Form bắt buộc trước live

1
Đăng ký tại formspree.io

Free plan: 50 submissions/tháng. Đủ để bắt đầu.

2
Tạo form mới → lấy Form ID

Dạng: xyzabc12 (8 ký tự)

3
Cập nhật action trong contact/index.html

Tìm dòng action="https://formspree.io/f/YOUR_FORM_ID"

Thay YOUR_FORM_ID bằng ID thật.

4
Newsletter form: dùng riêng một Form ID khác

Tạo form thứ 2 trong Formspree cho newsletter. Hoặc dùng Mailchimp embed.
