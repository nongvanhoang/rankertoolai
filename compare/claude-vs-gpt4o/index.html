Writing Quality

Across 10 writing tests (blog posts, emails, creative fiction, technical docs, and persuasive essays), Claude 3.5 Sonnet produced more natural, varied prose with better tonal awareness. Coding

On 8 coding tasks including Python scripting, React components, SQL queries, and debugging, GPT-4o (accessed via ChatGPT Plus with access to o3) outperformed Claude on complex algorithmic problems. Claude explained its reasoning more thoroughly, which is valuable when learning. For production coding where accuracy is the priority, GPT-4o's o3 access tips the scales.
